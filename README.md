Horkruks to system stworzony na potrzeby pracy inżynierskiej.

## Aby zacząć

Najpierw uruchom serwer developmentu:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Otwórz [http://localhost:3000](http://localhost:3000) aby zobaczyć wynik.
